## Description
In many parts of Ghana, and elsewhere in West Africa, a personâ€™s name is determined by the day of the week on which she/he was born This app generates an Akan name based on the day you were born. A user enters their birthday details and the application uses the a formular to generate the user's Akan name.
##Formular
Day of the week (d) = ( ( (CC/4) -2*CC-1) + ((5*YY/4) ) + ((26*(MM+1)/10)) + DD ) mod 7
